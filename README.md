#  DragonBall Advanced App

## Summary
In this project, I try to create an app to consumes from DragonBall API with local data cached by Room Database and implement dependency injection with Hilt.

## Key concepts
+ SOLID
+ Clean Architecture
+ Dependency Injection
+ ROOM
+ Hilt
